Install Stickies!

Follow these two easy steps to install Stickies!

Install Stickies! Javascript

Copy the following JavaScript and paste it just before the
closing </body> tag on your website:

<script src="http://websticki.es/stickies.js"></script>

Upload stickies.html to your Website

Simply upload stickies.html to the root of your website. Or create an HTML file called stickies.html and include the script below in the file. Then place it in the root of your site.

<html><head><script>isLoaded = true;</script></head><body></body></html>
